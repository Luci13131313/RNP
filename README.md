# Rock 'N' Pudgy

A rhythm game for **[Remix](https://remix.gg)** — tap to the beat.

🎮 **[Play it on Remix »](https://remix.gg/g/bf259375-6e03-4dae-8801-1372e1f9327d)**

## About
Hit the notes in time with the music. Ships with a built-in **beat-mapper** for charting tracks (`BeatMapper.html`).

## Tech
- Phaser 3 + TypeScript
- In-repo beat-mapper tool for authoring charts
- Integrated with the Remix SDK (leaderboard, game-over, mute)
